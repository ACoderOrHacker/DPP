### D++ Installation

#### Dependencies
- XMake >= 2.2.2
    * Optional, Compile-time only
    * See [xmake-io](https://xmake.io/)

#### Compile
```
xmake build -y

```

#### Package
```
xmake pack -y
```

#### Install
```
xmake install
```

#### Run
```
xmake run dpp --help
```
will run dpp to get help
